# アイコンフォルダ

実際のアイコンは**Canvas APIで動的に生成**されるため、静的な画像ファイルは不要です。`manifest.json`でも`icons`フィールドは指定していません。

## アイコン生成方法

`background.js`の`createIcon()`関数がリアルタイムでアイコンを生成します：

### 生成されるサイズ
- 16px
- 48px
- 128px

### デザイン
- **2×2グリッド**
- **左上3マス**: 予約状況（緑 or 赤+白バツ）
- **右下1マス**: オレンジ (#F57C00) + 白文字「一」

## 静的アイコンが必要な場合

Chrome Web Storeに公開する場合など、静的なアイコン画像が必要な場合は、以下のサイズの画像を手動で作成してください：

- `icon-16.png` (16×16px)
- `icon-48.png` (48×48px)
- `icon-128.png` (128×128px)

ただし、通常の使用では動的生成で十分です。
