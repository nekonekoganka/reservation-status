# ckreserve.com への自動データ取得 確認メールテンプレート

## 送信先

ChoiceRESERVE（ckreserve.com）のサポート窓口

## 件名

自院の予約空き状況データの自動取得について（許可確認）

## 本文

```
ChoiceRESERVE ご担当者様

お世話になっております。
貴社の予約システムを利用しております、○○眼科の○○と申します。

この度、自院のホームページおよび院内ディスプレイにて、
予約の空き状況をリアルタイムで患者様にご案内するシステムを
構築いたしました。

つきましては、以下の点についてご確認・ご了承をいただきたく
ご連絡いたしました。

■ 実施内容
・自院の予約ページ（ckreserve.com/clinic/○○○）から
  空き時間枠の情報を定期的に自動取得しております
・取得したデータは自院の患者様向けに表示する目的のみで使用しています
・他院の情報には一切アクセスしておりません

■ 技術的な詳細
・アクセス頻度: 診療時間帯は1〜5分間隔、夜間は5〜10分間隔
・取得する情報: 空き時間枠の一覧のみ（患者の個人情報は取得していません）
・サーバー負荷: 1回あたり通常のブラウザアクセスと同等の負荷です

■ お願い
・上記の利用方法について、問題がないかご確認いただけますでしょうか
・もし貴社のAPI等、より適切なデータ取得方法がございましたら
  ご教示いただけますと幸いです

ご多忙のところ恐れ入りますが、ご確認のほどよろしくお願いいたします。

─────────────────
○○眼科
院長 ○○ ○○
電話: ○○○-○○○-○○○○
メール: ○○○@○○○
─────────────────
```

## 補足

- `○○` の部分を実際の情報に置き換えてください
- ChoiceRESERVE が API を提供している場合は、スクレイピングから API に切り替えることで
  より安定した運用が可能になります
- 返信で許可が得られた場合は、メールを保管しておくことをお勧めします
