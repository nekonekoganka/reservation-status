# Chrome Web Store 掲載情報

## 拡張機能名
予約状況チェッカー

## 概要（短い説明）
予約ページの空き状況を確認するツール

## 詳細な説明

予約状況チェッカーは、オンライン予約ページの空き状況を簡単に確認できるChrome拡張機能です。

### 主な機能
- ボタンをクリックするだけで予約状況を自動チェック
- ページを自動更新して最新の情報を取得
- 予約可能な日時を素早く確認

### 使い方
1. 対象の予約ページを開く
2. 画面右下に表示される「🔄 予約状況を更新する」ボタンをクリック
3. ページが自動更新され、数秒後に予約状況が確認されます

### 特徴
- シンプルで使いやすいインターフェース
- 自動更新機能で最新情報を取得
- 個人利用に最適化
