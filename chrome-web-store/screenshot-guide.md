# スクリーンショット作成ガイド

Chrome Web Storeに公開する際、スクリーンショットが必要です。

## 必要なスクリーンショット

**必須:** 最低1枚のスクリーンショット

**推奨サイズ:**
- 1280 x 800 ピクセル
- または 640 x 400 ピクセル

**形式:** PNG または JPEG

## スクリーンショットの撮り方

### 方法1: 実際の画面をキャプチャ

1. 予約ページを開く
2. Chrome拡張機能のボタンが表示された状態でスクリーンショットを撮る
3. 画像編集ツールで 1280 x 800 にリサイズ

### 方法2: ダミー画像を使用（非公開公開の場合）

非公開公開なので、自分しか見ないため、以下のような簡単な画像でもOK：

1. 白い背景の画像 (1280 x 800)
2. 中央に「予約状況チェッカー」というテキストのみ

## ダミー画像の作成方法（簡単）

### オンラインツールを使う場合

1. **Canva** (https://www.canva.com/) を開く
2. カスタムサイズ: 1280 x 800 で新規作成
3. 白背景に「予約状況チェッカー」とテキストを追加
4. PNG形式でダウンロード
5. ファイル名を `screenshot1.png` として保存

### ペイント（Windows）を使う場合

1. ペイントを開く
2. キャンバスサイズを 1280 x 800 に設定
3. 白で塗りつぶし
4. テキストツールで「予約状況チェッカー」と入力
5. PNG形式で保存
6. ファイル名を `screenshot1.png` として保存

## スクリーンショットの保存場所

作成したスクリーンショットは、以下に保存してください：

```
/home/user/reservation-status/chrome-web-store/screenshot1.png
```

または、Chrome Web Store Developer Dashboard で直接アップロードすることも可能です。
